## Jupyter 快捷键操作

- 按下 Enter,进入 Edit 模式
- 按下 Esc,进入 Command 模式

**在 Command mode下：**

1. 运行当前 Cell，并移动到下一个 Cell
    - Shift + Enter

2. 创建 Cell
    - 按下 a ,在之前创建一个新的 cell
    - 按下 b ,在之后创建一个新的 cell

3. Cell 中 Code 和 Markdown 的切换
    - 按下 y, 进入 Code
    - 按下 m, 进入 Markdown

4. 显示 Cell 中的行数
    - 在一个 cell 中按下 l

5. 删除 Cell
    - 在一个 cell 中按两次 d

6. 保存 Notebook
    - 在一个 cell 中按下 s

7. 启动命令面板
    - 在一个 cell 中按下 Ctrl + Shift + P
    - 界面中的小键盘也可以启动

8. 设置标题格式
    - 按1，将单元格设定一级标题
    - 按2，将单元格设定二级标题
    - 按3，将单元格设定三级标题
    - 按4，将单元格设定四级标题
    - 按5，将单元格设定五级标题
    - 按6，将单元格设定六级标题
